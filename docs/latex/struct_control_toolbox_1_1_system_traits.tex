\subsubsection*{template$<$int State\+Dim = Eigen\+::\+Dynamic, int Input\+Dim = 0, int Output\+Dim = 0$>$\newline
struct Control\+Toolbox\+::\+System\+Traits$<$ State\+Dim, Input\+Dim, Output\+Dim $>$}
Defines dimensional properties and core types for dynamical systems. 


\begin{DoxyTemplParams}{Template Parameters}
{\em State\+Dim} & Dimension of system state (Eigen\+::\+Dynamic for dynamic-\/size systems) \\
\hline
{\em Input\+Dim} & Dimension of control input (0 for autonomous systems) \\
\hline
{\em Output\+Dim} & Dimension of system output (0 for state-\/only systems)\\
\hline
\end{DoxyTemplParams}
Provides compile-\/time dimensional checking and type-\/safe algebraic definitions for control system implementations. Integrates with Eigen for linear algebra operations.

\begin{DoxyNote}{Note}
All dimensions must satisfy\+:
\begin{DoxyItemize}
\item State\+Dim\+: Eigen\+::\+Dynamic or \texorpdfstring{$>$}{>} 0
\item Input\+Dim\+: \texorpdfstring{$>$}{>}= 0
\item Output\+Dim\+: \texorpdfstring{$>$}{>}= 0
\end{DoxyItemize}
\end{DoxyNote}
