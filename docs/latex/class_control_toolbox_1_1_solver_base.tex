\subsubsection*{template$<$typename Derived, typename System$>$\newline
class Control\+Toolbox\+::\+Solver\+Base$<$ Derived, System $>$}
CRTP base class for numerical integration algorithms. 


\begin{DoxyTemplParams}{Template Parameters}
{\em Derived} & Concrete solver implementation (CRTP derived class) \\
\hline
{\em System} & Type of dynamical system to solve (State\+Space\+System-\/based)\\
\hline
\end{DoxyTemplParams}
Provides common interface for\+:
\begin{DoxyItemize}
\item Time-\/stepping algorithms
\item \doxylink{class_control_toolbox_1_1_simulation}{Simulation} control mechanisms
\item Custom stop conditions
\item Configuration management
\end{DoxyItemize}

\end{DoxyNote}
