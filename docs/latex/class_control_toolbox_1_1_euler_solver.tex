\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename System$>$\newline
class Control\+Toolbox\+::\+Euler\+Solver$<$ System $>$}
Explicit Euler method numerical integrator. 


\begin{DoxyTemplParams}{Template Parameters}
{\em System} & Type of dynamical system to solve (State\+Space\+System-\/based)\\
\hline
\end{DoxyTemplParams}
Implements the forward Euler method\+:   \[y_{n+1} = y_n + h \cdot f(t_n, y_n)
\] where\+:
\begin{DoxyItemize}
\item $ h $ is the time step (dt)
\item $ f $ is the system derivative function
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Inherits from \doxylink{class_control_toolbox_1_1_solver_base}{Solver\+Base} using CRTP pattern 

While simple to implement, consider using higher-\/order methods (like RK4) for better accuracy in most applications 
\end{DoxyNote}
