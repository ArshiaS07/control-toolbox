Detailed Description

template<typename Derived, typename System>
class ControlToolbox::SolverBase< Derived, System >

CRTP base class for numerical integration algorithms.

Template Parameters
DerivedConcrete solver implementation (CRTP derived class)
SystemType of dynamical system to solve (StateSpaceSystem-based)

Provides common interface for:

  • Time-stepping algorithms
  • Simulation control mechanisms
  • Custom stop conditions
  • Configuration management
Note
Derived classes must implement step_impl()
