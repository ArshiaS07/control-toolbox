Detailed Description

template<typename Derived, typename Traits>
class ControlToolbox::StateSpaceSystem< Derived, Traits >

CRTP base class for state-space control system implementations.

Template Parameters
DerivedConcrete system implementation (CRTP derived class)
TraitsSystem traits defining dimensions and types (SystemTraits-based)

Provides complete signal processing chain:

  • Reference generation
  • Control computation
  • Disturbance injection
  • Dynamics calculation
  • Output processing
  • Data logging
Note
Derived classes must implement:
  • computeDynamics_impl()
  • computeController_impl()
  • computeReference_impl() (if HasOutput)
  • Other optional _impl methods as needed

Execute complete simulation step with signal processing.

Parameters
[in]tCurrent simulation time
[in]xCurrent state vector
[in]logEnable data logging (default: true)
Returns
StateVector State derivative vector

Implements full signal processing chain:

  1. Reference signal generation (if HasOutput)
  2. Previous output handling (one-step delay)
  3. Measurement noise injection (if HasOutput)
  4. Control signal computation (if HasInput)
  5. Input disturbance injection (if HasInput)
  6. System dynamics calculation
  7. Output update for next step (if HasOutput)
Note
Uses CRTP pattern to delegate implementation to derived class
