Detailed Description

template<typename Traits>
class ControlToolbox::SystemLogger< Traits >

Comprehensive data logging and analysis utility for dynamical systems.

Template Parameters
TraitsSystem traits type defining dimensional properties (SystemTraits-based)

Provides:

  • Time-series data recording at multiple signal points
  • Temporal query operations
  • CSV export/import capabilities
  • Basic signal analysis functions
Note
Requires C++17 or later for filesystem support
