Detailed Description

template<int StateDim = Eigen::Dynamic, int InputDim = 0, int OutputDim = 0>
struct ControlToolbox::SystemTraits< StateDim, InputDim, OutputDim >

Defines dimensional properties and core types for dynamical systems.

Template Parameters
StateDimDimension of system state (Eigen::Dynamic for dynamic-size systems)
InputDimDimension of control input (0 for autonomous systems)
OutputDimDimension of system output (0 for state-only systems)

Provides compile-time dimensional checking and type-safe algebraic definitions for control system implementations. Integrates with Eigen for linear algebra operations.

Note
All dimensions must satisfy:
  • StateDim: Eigen::Dynamic or > 0
  • InputDim: >= 0
  • OutputDim: >= 0

Example usage:

using DoubleIntegratorTraits = SystemTraits<2, 1, 2>;
