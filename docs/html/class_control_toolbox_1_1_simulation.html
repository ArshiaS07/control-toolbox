Detailed Description

template<typename System, typename Solver>
class ControlToolbox::Simulation< System, Solver >

Manages complete simulation execution and data collection.

Template Parameters
SystemDynamical system type (StateSpaceSystem-based)
SolverNumerical integrator type (SolverBase-based)

Coordinates the interaction between:

  • Dynamical system model
  • Numerical integration method
  • Data logging infrastructure
Note
Uses fixed-time stepping scheme
Maintains ownership of system and solver instances

template<typename System, typename Solver>
void ControlToolbox::Simulation< System, Solver >::run ( TimeType dt)
inline

Execute complete simulation.

Parameters
[in]dtFixed time step size (seconds)

Simulation loop steps:

  1. Check stop conditions
  2. Log current state through system derivative call
  3. Perform integration step
  4. Advance simulation time
Note
Logging is triggered via system's computeSystemDerivative
